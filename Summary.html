API:t

OpenWeather API

OpenWeather API antaa tämänhetkisiä säätietoja paikkakunnassa, jonka nimen käyttäjä syöttää hakukenttään

Koodiesimerkkejä:

                
                    const cityNameOutput = document.getElementById('cityName');

                    //api-kysely
                    fetch(`https://api.openweathermap.org/data/2.5/weather?q=${cityNameInput.value}&appid=cdda7f75f58e839c72c72d4cb3709bef&units=metric`)
                    .then(resp => resp.json())
                    .then(data => {
                        console.log(data);
                        let name = data['name'];
                        let temperature = data['main']['temp'];
                        let description = data['weather'][0]['description'];
                    })

                    .catch(error => console.log(error));
